# Yearly Review

Read the yearly plan and all quarterly/monthly/weekly files and any data in `metrics/`.

Append to `# Review`:
- **Goal Evaluation** — each goal: on track / at risk / off track, with evidence
- **Constraint Analysis** — single biggest limiter over the year
- **Hard Truth** — honest assessment of progress and effort
- **Adjustments** — up to 3 system or behaviour changes
- **Next Directive** — the one thing that should guide the coming year
