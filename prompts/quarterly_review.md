# Quarterly Review

Read the quarterly plan and all monthly/weekly files from the past 90 days.

Append to `# Review`:
- **Goal Evaluation** — each goal: on track / at risk / off track, with evidence
- **Constraint Analysis** — single biggest limiter over the quarter
- **Hard Truth** — honest assessment of focus, effort, and alignment
- **Adjustments** — up to 3 system or behaviour changes
- **Next Directive** — the one thing that should guide the next 90 days
